# Seasonal Time Converter
This tool is a webpage that displays a timestamp from one timezone in the user's local time, accounting for daylight saving time.

To use it, to go the link: `https://erallie.github.io/Arizona-Time-Converter/?t=XX:XX&tz=America/Phoenix` where:
- `XX:XX` is the time in 24-hour time.
- `America/Phoenix` is the IANA time zone identifier.
