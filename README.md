# Arizona Time Converter
This tool is a webpage that displays a timestamp from Arizona time in the user's local time.

To use it, to go the link: `https://erallie.github.io/Arizona-Time-Converter/?t=XX:XX` where `XX:XX` is the Arizona time in 24-hour time.

For now, this is only limited to Arizona. But in the future, I may change it so that it can use any time zone. Depending on popular demand.

Feel free to use whenever!
